# Snowing Central Park 3D

A high-fidelity, futuristic 3D interactive world of Central Park, featuring a "Ice & Glass" aesthetic.

## Features

*   **Futuristic NYC Skyline**: Towering, refractive crystal skyscrapers with warm interior lighting.
*   **Accurate Geography**: Central Park layout with a rectangular void surrounded by the city.
*   **Cinematic Atmosphere**: Twilight blue lighting, realistic reflections, and volumetric fog.
*   **Optimized Performance**:
    *   **Instanced Rendering**: Efficiently renders hundreds of buildings and trees using `InstancedMesh`.
    *   **GPU Snow Shader**: Handles 15,000+ snow particles on the GPU for smooth performance.
    *   **Optimized Lighting**: Tuned shadow maps and lighting calculations for high frame rates.

## Tech Stack

*   React 19
*   Three.js / React Three Fiber
*   React Three Drei (Environment, OrbitControls, Stars, Cloud)
*   React Three Postprocessing (Bloom, Vignette, ToneMapping)
*   Tailwind CSS 4
